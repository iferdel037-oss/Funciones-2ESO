# Funciones-2ESO: Recursos Interactivos LOMLOE

Recursos completos para la unidad "Funciones" de 2º ESO, alineados con la LOMLOE. Creatos con eXeLearning para que puedas explicar de forma visual e interactiva:
- **¿Qué es una función?**  
- **Funciones lineales y afines**  
- **Interpretación de gráficas**  
- **Problemas de la vida real**  

## 📚 Contenido detallado
1. **Unidad 1: Concepto de función**  
   - Variables dependientes/independientes  
   - Formas de representar funciones (tabla, gráfica, fórmula)  
   - Actividades interactivas iDevice  

2. **Unidad 2: Funciones lineales `y = mx`**  
   - Pendiente y ordenada en el origen  
   - Gráficas paso a paso  
   - Ejemplos: móviles, precios, distancias  

3. **Unidad 3: Funciones afines `y = mx + n`**  
   - Diferencias con la lineal  
   - Cómo calcular la ecuación a partir de 2 puntos  
   - Actividades autocorregibles  

4. **Unidad 4: Aplicaciones**  
   - Proporcionalidad directa  
   - Análisis de gráficas de la vida real (estadísticas, ofertas...)  
   - Proyecto final: Crea tu propia función  

## 🎯 Para qué sirve este recurso
- **Ahorra tiempo**: Tienes 4 unidades listas para Moodle, Classroom o exportar a SCORM.  
- **Alumnos motivados**: Actividades interactivas, vídeos y ejemplos cercanos.  
- **Adaptable**: Edita textos, añade ejercicios o quita partes.  

## 💻 Cómo usarlo
### Opción 1: Editar el .elp
1. Abre `Funciones-2ESO.elp` en eXeLearning  
2. Personaliza contenidos  
3. Exporta a SCORM/Web  

### Opción 2: Subir SCORM directo
1. Descarga `Funciones-2ESO_SCORM.zip`  
2. Súbelo a tu LMS (Moodle, Classroom...)  
3. Listo para que alumnos hagan actividades
